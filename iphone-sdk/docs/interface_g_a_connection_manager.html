Detailed Description

A class for communicating with Geo API servers. This is not thread-safe. If you want to send multiple API requests in parallel, you should instantiate a separate GAConnectionManager for each parallel request.

To get data from the GeoAPI, implement the GAConnectionDelegate protocol to handle the data returned from the GAConnectionManager request methods.

See also:
GAConnectionDelegate.
http://api.geoapi.com/ for full API details.
