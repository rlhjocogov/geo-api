Detailed Description

A protocol for handling Geo API connection events. See GAConnectionManager. If a request completes successfully, GAConnectionManager calls receivedResponseString:. If the request failed, it calls requestFailed:.

See also:
http://api.geoapi.com/ for full API documentation.
